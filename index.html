ПРАВИЛА СЕРВЕРА

1. Загальні правила

  • 1.1. Поважайте інших гравців та адміністрацію.
  • 1.2. Заборонено будь-які форми дискримінації.
  • 1.3. Заборонено використання читів та будь-яких програм, що надають перевагу.
  • 1.4. Нікнейм повинен бути адекватним, без образ та порушень.
  • 1.5. Заборонено рекламу інших серверів чи ресурсів без дозволу адміністрації.

2. Ігровий процес

  • 2.1. Заборонено гриферство та навмисне псування гри іншим гравцям.
  • 2.2. Заборонено використання багів та експлойтів.
  • 2.3. Teamkill дозволений лише у випадках самозахисту або за домовленістю.
  • 2.4. AFK більше 3 хвилин без попередження - причина для кика.
  • 2.5. Заборонено спам у чат чи голосовий чат.

3. VIP правила

  • 3.1. VIP статус не дає права порушувати правила сервера.
  • 3.2. VIP гравці можуть отримувати додаткові переваги, але не можуть зловживати ними.
  • 3.3. Передача VIP-акаунта третім особам заборонена.
  • 3.4. Адміністрація має право скасувати VIP статус за порушення правил без повернення коштів.

4. Накази адміністрації

  • 4.1. Накази адміністрації є обов'язковими до виконання.
  • 4.2. Спори з адміністрацією ведуться у приватному чаті або на дискорд сервері.
  • 4.3. Невиконання наказу адміністрації може призвести до блокування.
  • 4.4. Адміністрація має право застосовувати покарання на власний розсуд.

5. Санкції за порушення

  • 5.1. Перше порушення - попередження.
  • 5.2. Друге порушення - мут/кік з сервера.
  • 5.3. Серйозні порушення - бан від 1 дня до перманентного.
  • 5.4. Використання читів - перманентний бан без можливості розгляду.
  • 5.5. Адміністрація залишає за собою право змінювати строки покарань.

6. Додаткові правила

  • 6.1. Дотримуйтесь спільнотних норм поведінки.
  • 6.2. Заборонено використання нецензурної лексики у глобальному чаті.
  • 6.3. Поважайте рішення адміністрації.
  • 6.4. Заборонено використання сторонніх програм для отримання переваг.
  • 6.5. Будь-які спроби обійти правила будуть каратися баном.

7. Технічні вимоги

  • 7.1. Гравець несе відповідальність за свій акаунт.
  • 7.2. Проблеми з підключенням не є підставою для порушення правил.
  • 7.3. Заборонено використовувати баги карт.
  • 7.4. Технічні питання вирішуються через підтримку.
